# Nightfall Knight

Nightfall Knight is a 2D game currently in development, built using HTML, CSS, and JavaScript.  
The project focuses on fast-paced combat, animated characters, and a dark fantasy aesthetic.

---

## Project Status

In Development

This game is actively being worked on. Features, visuals, and gameplay mechanics are subject to change.

---

## Planned Features

Animated player character (Nightfall Knight)  
Sword combat and shield blocking  
Movement animations including idle, walking, and running  
Browser-playable using HTML and JavaScript  
Clean, simple UI  
Future sound effects and enemies  

---

## Technologies Used

HTML for game container and structure  
CSS for styling and visuals  
JavaScript for game logic and animations
